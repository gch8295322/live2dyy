# live2dyy

> 代码基本不是原创，是数位看板娘开发大佬的代码结合而成的产物。

# Introduction

国服最强WordPress看板娘。没有之一！

+ 集成度更高、效果更干净
+ 体积更迷你
+ 内置音乐播放器，歌单来源于网易云音乐。音乐播放器切换隐藏/显示。
+ 右击看板娘调出秘密通道，上面可以自定义链接
